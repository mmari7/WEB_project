Название проекта: WishMaker
Автор проекта: Маркова Мария 10Б
Описание идеи: Создать сайт, на котором можно будет искать товары с маркетплейсов и добавлять их в избранное (в вишлисты)
Некоторые из вишлистов могут быть публичными, поэтому пользователи могут по имени пользователя смотреть публиные листы других

Функционал написан в файле main.py c помощью flask. Для взаимодейстсвия с пользователем используются формы flask-wtf, которые лежат в папке
forms. Сами страницы написаны на html, с использованием css и bootstrap, код каждой страницы находится в папке templates.
Действия с базой данных реализованы с помощью SQLAlchemy и сопутствующих модулей и описаны в папке data.

Интерфэйс приложения понятный, на каждоой странице есть вспомогательные надписи и сообщения, если понадобятся. Есть меню
навигации, которое позволяет свободно перемещаться по сайту.

В качестве маркетплейса взят вайлдбериз (wildberries), с которого происходит парсинг товаров по категориям. Сначала получается
список всех возможных категорий и предоставляется пользователю. Далее пользователь выбирает нужные ему категории, и происходит парсинг
страниц, а с них уже товаров, которые загружаются пользователю на страницу (для этого требуется немного времени).
